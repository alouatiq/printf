# Custom printf Implementation

This project is a custom implementation of the `printf` function in C.

## Features
- Supports conversion specifiers: `%c`, `%s`, `%%`, `%d`, `%i`, `%b`

## Compilation
```sh
gcc -Wall -Wextra -Werror -pedantic -std=gnu89 *.c -o printf

## Usage
./printf
